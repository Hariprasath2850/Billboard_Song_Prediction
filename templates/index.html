Welcome to Song Prediction Suite

The Mainstream of this project is to predict Popular and Unpopular Songs and there are two buttons created as well. First let me explain the Upload song button, here we upload songs from localhost and this application will result as popular or not. This completely works on machine learning algorithm which we earlier used to predict for different models. The best model is chosen based on accuracy and applied here. About Spotify link, here we take a link from Spotify website directly and this application predicts if the song is popular or not popular. This also uses a machine learning algorithm as well.
